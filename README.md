# 📰 Fake News Detection using Machine Learning

A Machine Learning-based Fake News Detection system that classifies news articles as **Real** or **Fake** using Natural Language Processing (NLP) techniques and Scikit-learn.

---

## 📌 Project Overview

Fake news spreads rapidly across digital platforms, making it difficult to distinguish between genuine and misleading information. This project uses Natural Language Processing and Machine Learning to analyze news articles and predict whether they are **Real** or **Fake**.

---

## 🚀 Features

- Predicts whether news is Real or Fake
- Text preprocessing pipeline
- TF-IDF Vectorization
- Machine Learning classification
- Fast predictions
- Easy to use

---
